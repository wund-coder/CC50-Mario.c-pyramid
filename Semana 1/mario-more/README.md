#CC50 - Mario-more

Esse exercício é referente ao curso de Havard CS50, no Brasil CC50. 
É uma varíavel mais complicada do Mario-less. O usuário será solicitado um número entre 1 a 8, caso o número seja menor que 1 ou maior que 8, o usuário será questionado novamente sobre a altura.
Após um número válido ser inserido, o programa gerará duas pirâmides, uma de frente a outra com um vão entre elas, ambas possuindo a altura solicitada.
Esse programa foi feito na linguagem C.

Em comparação com o primeiro exercício do Mario, o Mario-less, a facilidade foi maior, afinal, só precisei imprimir dois espaço (o vão) e repetir o loop das pirâmides, gerando a segunda pirâmde.
Nesse exercício consegui compreender melhor as chaves {}, as quais passei a chamar de "pastas".

### Exemplo de execução:
Quando a altura inserida é `4`, o programa imprime:

   #  #
  ##  ##
 ###  ###
####  ####
